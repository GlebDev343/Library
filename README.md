#  Library Manager


## Описание

Консольное приложение для управления библиотекой книг.


## Функционал

1. Добавление книги.
2. Удаление книги.
3. Поиск книги.
4. Отображение всех книг.
5. Изменение статуса книги.

## Установка и запуск

1. Склонируйте репозиторий.
2. Убедитесь, что установлен Python 3.10+.
3. Запустите приложение:
   ```bash
   python main.py
